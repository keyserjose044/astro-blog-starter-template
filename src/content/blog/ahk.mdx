# ⌨️ Building a One-Handed Productivity System with AutoHotkey

In January 2026, I spent 41 hours building a productivity system that runs almost entirely from my left hand on the keyboard. No mouse. No clicking through folders. No retyping the same phrases.

Just 16 simple AutoHotkey hotstrings, all ending in 'xx'.

With those alone, I can:

- Instantly arrange my workspaces  
- Teleport into project folders  
- Launch full workflows  
- Paste smart boilerplate with live inputs  
- Trigger advanced ChatGPT prompt menus  
- Summarize conversations in one keystroke  

This post breaks down the breakthroughs behind the system and how anyone can build something similar.

---

## 🧠 The Core Idea: One Hand, Infinite Workflows

Every hotstring in my system ends with:

<pre><code>xx</code></pre>

So instead of remembering dozens of shortcuts, I only memorize:

**16 easy prefixes + xx**

These being:

- **qxx** — *"Qore" Layout*  
  “Core” but with a Q. Opens 4 apps (customizable: ChatGPT, Obsidian, Firefox, File Explorer) in a custom layout across 2 screens.

- **wxx** — *"Work" Layout*  
  Opens 4 apps (Word, Excel, PowerPoint, OriginPro) with layout.

- **exx** — *"Email" Layout*  
  “Communications” — Teams, Outlook, Thunderbird, Phone Link + layout.

- **rxx** — *"Run" Menu*  
  Catchall launcher for browser + website, apps, etc.

- **txx** — *"Teleport" Menu*  
  Launcher for folders and app combos across your desktop.

- **yxx** — *"Why" Menu*  
  Explains all 16 shortcuts.

---

- **axx** → Types: *"Please let me know if anything else is needed."*  
- **sxx** → Types: *"Please give a super brief, no-fluff bullet summary of this entire chat. No extra commentary."*  
- **dxx** → Inserts timestamp  
  - Example: `Mon, Mar 30, 2026 — 3:44 PM`  
- **fxx** → Types: *"Please give a super short response. 1–2 sentences max. No prefacing."*  
- **gxx** → Types: *"Thank you for your time and consideration."*

---

- **zxx** — *"Zearch" Prompt Menu*  
  PhD prompts for ChatGPT ready to use.

- **xxx** — *Catchall Prompt Menu*  
  General-purpose prompts.

- **cxx** — *"Code" Prompt Menu*  
  Prompts specifically for coding with ChatGPT.

- **vxx** — *"Vault"*  
  Custom long-form ChatGPT prompt for working with research papers.

- **bxx** — *"Blessings"*  
  Input:
  - 3 things you're grateful for  
  - 2 things you want to avoid  
  → Outputs a full prayer including these.

Your left hand never leaves the keyboard. Your brain stays in flow. And everything becomes muscle memory.

## 🧩 The 4 Pillars of My AutoHotkey System

---

### ⚡ 1. Layout Arrangers & App Openers

One command can open ChatGPT, Obsidian, my code editor, and file explorer, then snap everything into position across monitors.

Instead of spending 2 to 5 minutes setting up each morning, it happens instantly.

This alone saves me hours every week.

---

### 📁 2. Folder & Workflow Teleporters

I don't browse directories anymore. I type:

<pre><code>projectxx</code></pre>

And I'm instantly inside the exact folder I need.

No more:

<pre><code>Documents -> PhD -> Scripts -> Version 4 -> Output -> Final -> MaybeFinal -> ActuallyFinal</code></pre>

Just teleport.

---

### ✍️ 3. Smart Phrases (Tiny but Powerful)

I reuse certain phrases constantly. For example:

> Please give a super brief, no-fluff bullet summary of this entire chat. No extra commentary.

Instead of retyping it every time, one hotstring pastes it instantly.

It sounds small, but repeated hundreds of times, it saves real hours.

---

### 🤖 4. ChatGPT Prompt Menus (The Game Changer)

This is where things truly leveled up.

I built interactive AutoHotkey menus that:

- Pop up with structured options  
- Ask for inputs  
- Generate full prompts automatically  

Examples include:

- Summarization menus  
- Code refactor workflows  
- Analysis prompts  
- Brainstorming assistants  

It's like having a personal AI control panel.

No remembering prompts.  
No messy typing.  
Just choose -> fill -> send.
